# Direccion visual, lore y assets IA

## Fantasia central

El jugador descubre un cristal dormido que despierta una fabrica cosmica. Cada etapa convierte el laboratorio en algo mas grande: taller, portal, fundicion orbital y motor galactico.

## Tono

- Misterioso al inicio.
- Satisfactorio y brillante al tocar.
- Cada desbloqueo debe sentirse como expansion de escala.
- El jugador no solo compra upgrades: reconstruye una tecnologia antigua.

## Etapas visuales

1. Fragmento Dormido
   - Laboratorio oscuro.
   - Cristal pequeno con brillo cian.
   - Pocos efectos.

2. Primera Resonancia
   - Mesa iluminada.
   - Chispas suaves.
   - Herramientas activas.

3. Taller Automatico
   - Maquinas pequenas.
   - Cintas, brazos, mini robots.
   - Mas movimiento.

4. Nucleo del Portal
   - Portal energetico detras del cristal.
   - Colores violeta/cian.
   - Sensacion de salto dimensional.

5. Fundicion Orbital
   - Fondo espacial.
   - Drones y fabrica en orbita.
   - Recompensas mas vistosas.

6. Motor Galactico
   - Reactor enorme.
   - Galaxia al fondo.
   - El cristal original parece una llave cosmica.

## Reglas de consistencia

- Mantener paleta cian, azul oscuro, dorado y acentos magenta/violeta.
- No usar imagenes con texto generado dentro.
- Exportar sprites UI con fondo transparente cuando corresponda.
- Priorizar siluetas claras sobre detalle excesivo.
- Optimizar imagenes antes de subirlas a GitHub.

## Primera prueba visual

Para la primera prueba basta con:

- 1 cristal principal.
- 1 fondo de laboratorio.
- 3 iconos de upgrade.
- 1 icono de recompensa diaria.
- 1 icono de anuncio recompensado.

El resto puede seguir con UI generada por codigo hasta validar que el loop engancha.
